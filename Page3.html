The style tags

Before we even start using CSS we have to introduce a new tag from html. This tag is the style tag. The style tag is a html tag which is the main way to input CSS onto your HTML code. The way you implement the style tag is by using the style opening tag style, putting your code into the tag for it to function, and then the style closing tag .

Colours

Colours in CSS are actually one of the most confusing parts to learn in CSS because they are the first things you learn and also have so many different functions. Background-color, color commands, text-color, and so much more which makes it quite a pain to learn, in this chapter all I am going to teach are these three but there are more so feel free to explore in the cheat sheet or in later chapters.

Colour commands

Colours in CSS are very simple and are quite easy to learn. To implement a colour to a command all you have to do is write the color name. So, for example, if I wanted my text to be blue then all I would have to do is type: text-color: blue; But you can also use the specific code name for that colour so you can get the exact colour you want. All you have to do to find a name for a colour in html and CSS is to go into your browser and type “colour-name CSS defining code.”

Background-color

The background color command is usually placed inside the style tag and inside the body subsection in the style tag. The background color command is commonly used to give color to the base of your web page. But it can also be used to give a color to a tag. For example, you can use background color to change the color of a your webpages base color.

Text colour

The text colour command is a piece of code that is put into the style tag for which tag you want to make a different colour on your web page. The text colour command is used by picking a specific colour and then coding to have your text be that colour. For example, if you want all of your h1 tags to be blue you would code: h1{colour: blue;} and then all of your h1 tags in your web page would be command to have blue text.

What is CSS used for?

The main use for CSS is to expand on a html element. CSS is not intended to be by itself. The main use for CSS is to go off a html element and had visual assistance to that element. So, for CSS the main way to add it to html is by using the style tag so make sure you are still using it like how I described how to use it in the colour section. Now I am going to be teaching a text alignment, fonts, font styles, and font sizes.

Text alignment

Text alignment is one of the easier CSS command with all you have to do to set it up is type in you text tag in to the style tag. Ex: p{}. And the and in the middle of those brackets: text-align center; this will make so your text will be in the center of your web page but there are many more alignments so feel free to explore.

Font

Fonts in CSS are also very easy with the same formula of writing the text tag into the style tag and then writing the command in the brackets and this time you are going to want to write “font-family:(ex: arial);” to make each piece of writing under that tag the font you selected it to be.

Font sizing

Font sizing in CSS is also the same as the font tag but instead of writing “font-family:;” you have to write font-size:_pt or_px; to determine the size you wany each text using that tag.

Font styling

Font styling is also has the exact same formula and the reason I am saying this is so you don’t forget because every little thing in coding matters and one mistake can ruin a project until it is fixed. So all you have to do for font styling is write font-weight, font-style, or font-variant for what you want to do. Font variant does things like all lowercase letters with the command “small caps”. Font weight does things like making your writing bold with the tag “bold”. And lastly. Font style does many things like making you text italic with the command “italic”.
