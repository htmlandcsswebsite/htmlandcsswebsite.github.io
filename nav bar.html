Part 2: Intro to Html and CSS

What is Html?

Html stands for Hypertext mark-up language. Html is a scripting language to help web development to make it easier to code a website with premade functions. Just to clarify, Html is not a programming language. It is a scripting language. It has premade functions and programs instead of ones you make yourself. HTML is used in web development to set up a base and instructions for the website.

What is CSS?

CSS stands for cascading style sheets. CSS is an extension for Html to show code on a website. CSS is used to style a website to make it different. CSS is primarily used for things like formatting. But it can also change colors, style where everything is, present text in many different ways like borders and CSS cards, and so much more.

Why use Html and CSS?

HTML and CSS are the primary website creation tools for aspiring web developers with easy-to-use functions and many ways to display their websites. Reasons to use Html and CSS are things like how it is one of the best beginner friendly web development tools, having almost infinite different ways to display information, the premade tags so you don’t have to program your own, and so much more which is why this is one of the most popular coding languages.

How to download MSVSCODE

All you need to do to download MSVSCODE is search up MSVSCODE download, and they should pop up the website for downloading. After it downloads click on the app icon to start up the software to start coding.
