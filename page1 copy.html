Part 1: What are html tags and what do they do

What are tags and how do they work in html?

Tags in HTML are premade programs that define what you are trying to do. Tags are how you change your website and make it have or do things. Tags can do drastic things and make a website itself. An example is how tags are the main ways to communicate using words. There are many tags in HTML letting you do almost anything you can think of if you know how they work. In this part we are only going to go over the basic tags but in later sections we will be using a vast array of tags so make sure you understand what they do and how they work.

How do tags work?

Tags in HTML are simple and only have a few rules. HTML tags consist of a opening command which opens a program. The command which has the program do the action you are commanding it to do program ... And the closing command which signals the end of a piece of code . once you have made that, than you are going to write what you want to do. Part 2: Basic tags in HTML and what they do

The basic tags of HTML

In this part we are going to be going over 6 tags. The first one being-

The HTML tags

The HTML tags are usually premade, and they are very simple. All they do is compile all of the programs you are making into one tag to make them work together. All you must do is put all your code into this tag so that they all work accordingly.

The Head tags

The Head tags are also usually a premade tag in a coding software, and it is one of 2 main sections in the HTML tag to put code in. This tag is not as important for websites, but you can do some styling for your website using this tag. The head tags are used to store data and is many used for things like logins or search engines to store what it knows for the user experience to do things like login or find information on a search engine.

The Body tags

The Body tags are the second main sections in the HTML tag to put your code in. This tag is used much more than the head tags and it is used to input information and put it onto the web page. The body tags are very diverse and are used to do things like write, format and style your web page with various different sub sections on this main section

The p tags

The p tags stand for paragraph tags, and they do just that. The p tags are very straightforward, being how all they do is allow you to type full paragraphs to display on your web page.

The h1 to h6 tags

The h1 to h6 tags are used to display titles and headings in a web page. The h1 to h6 tags are used by having the h1 tag being the biggest font size and it gradually goes all the way down to h6 which is the smallest font size.

The hr and br tags

The hr and br tags are used to indicate line break. These tags are meant to be placed in between the paragraphs that you want to have separate. These tags are also different from the others because instead of you having to use an opening and closing tag for it to operate. These tags only use the opening tag to display the line breaks. The differences between a hr and br tag is that the br tag is just a blank space and the hr tag adds a line to fill up that space but otherwise they are very similar.
