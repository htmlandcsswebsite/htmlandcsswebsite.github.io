Section 4

Formatting you web page

Part one:

The div tag

The div tag is the main tag to use when you want to format a website. The reason for this is because the div tag is used to create columns and rows that are perfect to turn boring websites to ones with nice sectioning and on that I easy to go around like this one hopefully.

How to use the div tag to section a website (HTML)

The first thing you need to do to create a great format for a website is with html. This section will be focusing on the sectioning I have made for my website, but this tutorial will be very self-explanatory so feel free to experiment. First what you have to do is type in the body tag: div class= “container or parent” div This is the container for you div tags and will consist of all of your formatting so make sure you put your div tags for you formatting in this tag or else your code will not work. Now after you have done that you have to do the exact same thing inside the opening and closing tag but instead of parent you want to write header, content, sidebar, and footer. This is the end of the html section be make sure you understand because if you move on to the CSS section and your html code is not correct then your code will not work.

Part two:

The CSS coding

Defining rows and columns

This is where formatting becomes tricky so make sure you are paying attention. Formatting in CSS is reliant on the head and style tags so make sure you put in code accordingly. The first thing you are going to want to do is define your rows and columns by giving creating a coresponding .parent or .container section in the style tag and add: grid-template-rows:; grid-template-columns:; to add your rows and columns into your code.

Grids and viewing size

In one of these sections you are going to have to define the area of the page that you want the format to take up by writing “height: vh100;” and “display: grid;” which will let you have your format take up 100% of the page and will let you have the section spread apart for how big your page will be.

Areas

After that you are going to want to write “grid template areas:” in the parent section in the style tag which sections all of you div tags starting. After you do that you are going to want to write “header header”, “content sidebar”, and “footer footer” in the grid template section. Then you are going to want to create 4 tags in the style tag for the 4 sections and write “grid area:” in each of them to link them to the grid template code an write the designated section in each of them. Ex: sidebar. = grid area: sidebar; which will connect all of your code and will make all of your div tags sectioned for your website.

Lastly you can add the grid-template-row and grid-template-column codes into the parent code to change the length and width of you sections. Ex: grid-template-row: 10% 30% 60%; for the 3 section tall format.

This is the end of this chapter of my website. I know this is one of the hardest parts so make sure you are familiar with what you are doing but if your are happy with your format then you can go on the next section.
