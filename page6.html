Section 6 Creating a navigation bar and links

Part 1:

Navigation bar set up:

The navigation bar is more of the same that we have already done where all you have to do is write in the div class of the section you want your navigation bar to be in and then write div class nav and write what you want that to navigate your web page to for how many things you want your navigation bar to have.

Setting up links

Links are trickier in to code in html about are more confusing than hard to do. The first part of a link in html are the ul tags. After you do that then you have to add li tags. Then you have to write the a tag which is the tag that actually sends you to the link. After you do all of that you are going to write herf=”link” target=”_blank” inside the opening a tag to create a link in html.

Styling

Styling your navigation bar is not really that advanced where all to can really do is align your items to the left or right and if you want a horizontal navigation bar you add is display inline in the style tag for the li tag. And if you want a vertical navigation bar you just keep it the same. But if you want to learn animations and transitions then you should go to section 8 where I go in depth about animation in CSS.
