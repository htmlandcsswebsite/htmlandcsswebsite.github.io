Section 7 CSS cards and Margins

Part One

CSS cards

What are CSS cards

CSS cards are cards made in CSS that visually make your website look cleaner in a box for, but they also help navigate by if you click on one it will transfer you to another web page. A great example of this are online shopping websites where each item is a CSS card that you click on to add to go to a description of that item. These can be used in many ways, but the main ways are for navigation and organization.

Html

Again, all you must do in html to create a CSS are write div class=”card”> and then write your information in the middle. Which is just more of the same.

CSS

CSS is also very easy to do where all you have to do is write in the style tag .card. Then you are going to want to right down the width you want your card to be. Then you are going to want to write image in the .card and write 100% which will make your image as big as it can be to fit in your card. Yes it is really that easy but adding fancy stuff like animations takes time so make sure to play around with CSS cards.

Part two

Margins, border and padding

What are margins, border, and padding

Margins are the space around items. For example a 10px margin would spread two paragraphs 10px away from each other. Borders are just outlines of your content in a box format. And padding is the space between your content and a border.

How do you add margins, borders, and padding

These are also very easy in which all you have to do is write margin, border, or padding to the style tag and in which item you want it to apply to and then just write the distance in px measurements.
